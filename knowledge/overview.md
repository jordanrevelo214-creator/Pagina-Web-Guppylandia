# GuppyLandia - Proyecto Overview

## Descripción
GuppyLandia es una tienda online especializada en acuariofilia (peces, plantas naturales y accesorios premium) ubicada en Tulcán, Ecuador. El objetivo es transicionar de una estructura estática legacy a una plataforma moderna, rápida y visualmente impactante.

## Objetivos del Proyecto
1.  **Modernización:** Migrar de HTML/CSS estático a Astro v6 para mejor rendimiento y mantenibilidad.
2.  **Diseño Pro:** Replicar un diseño "IA-Generated" premium con una estética acuática, limpia y fluida.
3.  **Funcionalidad:** Implementar gestión de inventario real, sistema de carrito y contacto directo vía WhatsApp/Email.
4.  **Escalabilidad:** Estructura preparada para integrarse con Cloudflare D1 (Base de Datos) y Cloudflare Pages.

## Público Objetivo
Acuaristas principiantes y expertos que buscan calidad, asesoría y productos premium para sus acuarios.
